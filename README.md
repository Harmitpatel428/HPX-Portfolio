# HPX Eigen | Intelligent B2B CRM Portfolio

![HPX Eigen Logo](Harmit.png)

A premium, high-performance founder portfolio and product showcase for **HPX Eigen** — an intelligent B2B CRM engineered for mission-critical growth.

## 🚀 Overview

This project is a state-of-the-art landing page and dashboard visualization designed to exhibit the capabilities of HPX Eigen. It features a modern, dark-themed aesthetic with glassmorphism effects, fluid animations, and a fully responsive layout.

## ✨ Key Features

- **Hero Experience**: Dynamic dashboard visualization with Mac-style window chrome.
- **Milestones & Momentum**: A structured roadmap showing the evolution from 2024 to 2026 Launch Readiness.
- **Intelligent CRM Visuals**: Interactive Kanban boards, revenue charts, and live feed simulations.
- **Custom Plan Builder**: A comprehensive lead generation system with advanced form validation and success states.
- **Responsive Design**: Optimized for everything from high-resolution desktops to mobile devices.
- **Performance First**: Built with vanilla HTML, CSS, and Tailwind for maximum speed and SEO efficiency.

## 🛠️ Tech Stack

- **Core**: HTML5, Semantic Markup
- **Styling**: Tailwind CSS (v3), Vanilla CSS for custom animations
- **Icons**: Material Symbols Outlined (Google Fonts)
- **Typography**: Public Sans (Google Fonts)
- **Interactions**: Vanilla JavaScript (ES6+)

## 📦 Getting Started

To view the project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Harmitpatel428/HPX-Portfolio.git
   ```
2. Open `index.html` in your preferred browser.
